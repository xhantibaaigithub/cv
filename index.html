About Me

I am a passionate developer with a particular interest in frontend development. My experience with frontend development spans across various programming languages including Javascript, Typescript and Dart, within which I have developed my skill set in the Lit and React libraries and the Flutter and Vue frameworks. My backend programming proficiency includes Node.js, Quarkus, Python, Firebase and SQL. I have also developed my skill in automated testing using the Cypress testing framework for end-to-end testing.

My professional career has afforded me opportunities such as the most recent being as an intermediate full stack developer at BMW Group where I was part of a team of developers creating and implementing sophisticated web application software servicing an international client base using Lit.js for frontend development, Quarkus for backend development and Cypress for end-to-end testing. In my role I was also introduced to other technologies such as Kafka, and ElasticSearch.

In my previous role at Mango Airlines as a Senior QA and Support Analyst, my responsibilities centered around business analysis where I drafted functional specifications, created user stories that would be worked on by developers during each sprint using Agile methodology, and conducted manual testing. I also provided second-level application support, website content administration and used tools such as Google Analytics to monitor and analyze user behavior, then make recommendations on how improvements can be made to minimize the bounce rate and optimize the sales conversion rate to increase revenue. This experience provided valuable insights for enhancing user experience and optimizing marketing strategies using data-driven decision-making.

My experience with Flutter is limited to a project I have embarked on creating a social media application, and my experience with React is with an attempt at creating a reservation system, both of which leverage Firebase and Node.js for backend functionality. Both projects have been purely to expand my programming language proficiency, which further exhibits my interest in frontend development.
